# Password Breach Checker

This is a simple Python project that checks whether a password is present in a list of commonly used passwords.

I created this project to understand why weak and frequently used passwords can be a security risk.

## Features

- Checks passwords against a common password list
- Warns users about weak passwords
- Beginner-friendly implementation
- Demonstrates basic password security concepts

## Example

Input:
123456

Output:
Warning!
This password is commonly used and may be unsafe.
